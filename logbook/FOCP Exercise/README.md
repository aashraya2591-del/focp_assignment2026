What these Documents are 
These are completed exercise logbooks for the Fundamentals of computer programming (FOCP) course. Each document contains questions and answers covering specific python programming topics studied throughout the course weeks.
Topics Covered
Week 1: Introduction to programming
•	Programming language basics (Python)
•	Input -Processing – Output model
•	Programming language generations
•	Compilers and interpreters
•	REPL (Read Eval Print loop)
•	Errors types (syntax vs logical)
Week 2: Variables and Types
•	Variables and data types
•	Type conversion and dynamic typing 
•	Input/Output operations
•	String operations and slicing
•	Lists and mutability
•	Functions and arguments 
Week 4: Functions
•	Functions definitions and calls
•	Importing modules and libraries
•	Parameters and arguments
•	Return statements and values
•	Default arguments
•	Variable length arguments 
•	Lambda functions

Week 5: Scripts and Modules 

•	Python scripts (py.files)
Running programs from command line
•	Command line arguments (sys.argv)
Import statements and aliases
•	Symbol tables and namespaces
•	Modules vs script execution 
•	The name variable 

Week 6: lists and tuples
•	List methods (append, extend, pop, reverse, etc)
•	Mutators vs accessors
•	List comprehensions
•	Tuples and immutability
•	Tuple unpacking
•	Sequence operations 

Week 7: Sets and Dictionaries 
•	Set operations and methods
•	Set vs list differences
•	Set comprehensions
•	Frozen sets
•	Dictionary structure (Key value pairs)
•	Dictionary operations and method 

Week 8 : Formatting and file handling
•	F-Strings and string formatting
•	Format specifiers
•	File operations (open, read, write, close)
•	File modes (r,w,r+,b)
•	Reading methos (read, redline, read lines)
•	File iteration
•	With statements for file handling 

Purpose
•	These logbooks serve as
•	Study materials for reviewing Python concepts
•	Reference documents for syntax and examples
•	Test preparation resources (allowed during week 6 test)
progress tracking of course completion
